## 🚀 БЫСТРЫЙ СТАРТ (5 МИНУТ)

### Шаг 1: Создайте таблицу в Supabase

Откройте **SQL Editor** и скопируйте:

```sql
-- Создание таблицы для анкет клана TIGER KING OLD
CREATE TABLE IF NOT EXISTS applications (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL,
  nickname TEXT NOT NULL,
  age TEXT,
  experience TEXT,
  rank TEXT,
  discord TEXT,
  about TEXT
);

-- Включаем доступ на чтение и запись
ALTER TABLE applications ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Allow public insert" ON applications
  FOR INSERT WITH CHECK (true);

CREATE POLICY "Allow admin read" ON applications
  FOR SELECT USING (true);
```

Нажмите **Run** ✓

### Шаг 2: Обновите .env.local

```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
ADMIN_PASSWORD=TigerKingOld2024
```

### Шаг 3: Запустите локально

```bash
npm install
npm run dev
```

Откройте http://localhost:3000/form

### Шаг 4: Тестируйте

1. Заполните форму:
   - Ник: `TestPlayer`
   - Возраст: `20`
   - Опыт: `2 года`
   - Ранг: `Gold 3`
   - Discord: `TestPlayer#1234`
   - О себе: `Ищу серьезный клан`

2. Нажмите "Отправить заявку"

3. Откройте `/admin` (пароль: `TigerKingOld2024`)

4. Должна быть видна ваша заявка!

## ❓ ЧАСТЫЕ ВОПРОСЫ

**Q: Что если я хочу добавить новое поле?**
```sql
ALTER TABLE applications ADD COLUMN new_field TEXT;
```
Затем обновите форму и админ-панель в коде.

**Q: Как изменить пароль админа?**
В `.env.local` измените:
```env
ADMIN_PASSWORD=mynewpassword123
```

**Q: Как экспортировать все заявки?**
В Supabase → Table Editor → `applications` → Download (CSV)

**Q: Как удалить все заявки?**
В SQL Editor:
```sql
DELETE FROM applications WHERE true;
```
