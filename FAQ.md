# ❓ FAQ - Часто задаваемые вопросы

## 🎮 Для игроков

### Как подать заявку?
1. Откройте страницу формы (`/form`)
2. Заполните все поля (отмеченные звездочкой * обязательны)
3. Нажмите кнопку "Отправить заявку"
4. Дождитесь сообщения об успехе

### Какие данные вы собираете?
Мы собираем:
- Личную информацию (ник, ID, возраст, страну/город)
- Игровые навыки (ранг, K/D, реферируемую роль)
- Опыт в кланах и турнирах
- Контактную информацию (Discord/Telegram)
- Мотивацию присоединиться

Все данные защищены и используются только для набора в клан.

### Чему равна звездочка (*) на форме?
Звездочка обозначает обязательное поле. Вы не сможете отправить форму, если оне заполните ВСЕ обязательные поля.

### Почему форма не отправляется?
Возможные причины:
1. Вы не заполнили одно из обязательных полей (*)
2. Ошибка интернет соединения
3. Проблемы с сервером (проверьте позже)

Если ошибка повторяется, откройте браузер DevTools (F12) и посмотрите коне в консоли.

### Как долго обрабатывается заявка?
Обычно администраторы клана рассматривают заявки в течение 1-2 дней. Они свяжутся с вами через Discord/Telegram.

### Что если я неправильно заполнил поле?
Вы можете заполнить форму еще раз. Каждая отправка - это новая заявка. Администраторы будут рассматривать последнюю.

### Мой Discord/Telegram не прошел валидацию
Убедитесь, что введен в правильном формате:
- Discord: `YourName#1234` или `@YourName`
- Telegram: `@username` или `+79XXXXXXXXX`

### Сколько часов занимает заполнение формы?
Обычно 5-10 минут.

### Я забыл свой ID в PUBG, где его найти?
1. Откройте PUBG в игре
2. Откройте свой профиль
3. Скопируйте User ID (обычно в формате: `accountid123456` или `uid123456`)

Или посетите [pubg.com/profile](https://pubg.com) → Sign In → View your ID

### Могу я изменить свою заявку после отправки?
К сожалению, нет. Но вы можете заполнить новую форму с исправленной информацией.

## 🔐 Для администраторов

### Как войти в админ-панель?
1. Откройте `/admin`
2. Введите пароль: `TigerKingOld2024`
3. Нажмите Enter

Пароль можно изменить в файле `.env.local` (переменная `ADMIN_PASSWORD`).

### Я забыл пароль, что делать?
1. Откройте файл `.env.local` в вашем проекте
2. Найдите строку `ADMIN_PASSWORD=TigerKingOld2024`
3. Измените пароль на новый
4. Если приложение на Vercel, обновите переменную окружения в Settings → Environment Variables
5. Переразворните приложение

### Как просмотреть заявку?
1. В админ-панели нажмите на заявку в таблице
2. Справа откроется подробная информация
3. Вы можете прочитать все ответы игрока

### Как удалить заявку?
1. Нажмите на заявку в таблице
2. Справа нажмите кнопку "Удалить заявку"
3. Подтвердите удаление

**Внимание**: Удаление необратимо!

### Как скачать все заявки?
1. В Supabase Dashboard откройте **Table Editor**
2. Выберите таблицу `player_applications`
3. Нажмите кнопку **Download** (три точки вверху)
4. Выберите формат CSV
5. Откроется скачивание файла

### Как отправить заявку игроку?
Приложение не имеет встроенной функции отправки сообщений. Вы должны:
1. Скопировать контакт игрока (Discord/Telegram)
2. Связаться с ним лично через соцсети

### Сколько людей может быть администраторских в админ-панели?
Теоретически - много. Но все они используют одного пароль. Для более сложной системы авторизации потребуется дополнительная разработка.

### Можно ли изменить поля в форме?
Да, но нужно:
1. Отредактировать компонент формы (`components/RecruitmentForm.tsx`)
2. Обновить тип данных (`lib/supabase.ts`)
3. Добавить колонку в Supabase (SQL)
4. Переразворнуть приложение

### Можно ли изменить дизайн?
Да! Дизайн использует Tailwind CSS. Отредактируйте:
- Цвета в `tailwind.config.js`
- CSS в `app/globals.css`
- JSX в компонентах

### Как получить больше статистики?
В админ-панели показаны основные статистики. Для более подробной аналитики:
1. В Supabase Dashboard откройте **Settings** → **Extensions**
2. Включите Supabase Extensions для аналитики
3. Используйте SQL запросы для кастомных отчетов

### Можно ли экспортировать данные?
Да:
1. В Supabase Table Editor выберите таблицу
2. Нажмите **Download** → **CSV**
3. Откроется скачивание

Или через SQL:
```sql
SELECT * FROM player_applications;
```

### Как сделать бэкап данных?
Supabase автоматически делает бэкапы. Но для дополнительной безопасности:
1. Регулярно скачивайте CSV файлы
2. Храните их в надежном месте (Google Drive, Яндекс.Диск, и т.д.)

### Как восстановить данные?
1. В Supabase Dashboard → **Backups** (в Settings)
2. Выберите дату, на которую хочется восстановить
3. Нажмите **Restore**

### Как отключить форму и скрыть приложение?
1. Отредактируйте главную страницу (`app/page.tsx`)
2. Закомментируйте или удалите ссылку на форму
3. Или используйте `redirect()` для перенаправления

### Безопасно ли мое приложение?
Уровень безопасности зависит от:
- RLS политик в Supabase (контролируются в БД)
- Пароля админ-панели (лучше генерируйте сильный пароль)
- HTTPS соединения (обеспечивается Vercel или другим хостингом)

**Рекомендации**:
1. Используйте сильный пароль (минимум 12 символов)
2. Не делитесь пароля админом
3. Регулярно проверяйте logs на Supabase
4. Используйте двухфакторную аутентификацию на Vercel и Supabase

## 🐛 Решение проблем

### Ошибка: "Could not connect to Supabase"
**Причины**:
- Неверный URL или ключ API
- Суpabase проект отключен
- Интернет соединение
- Проект удален

**Решение**:
1. Проверьте `.env.local` или Vercel Environment Variables
2. Проверьте, что Supabase проект активен в dashboard
3. Проверьте интернет соединение
4. Контакт Supabase support

### Ошибка: "Table 'player_applications' does not exist"
**Причины**:
- SQL скрипт не был выполнен
- Таблица была задалена

**Решение**:
1. Откройте Supabase SQL Editor
2. Скопируйте и выполните SQL из `SETUP_SUPABASE.md`
3. Обновите приложение

### Админ-панель показывает пустой список
**Причины**:
- Заявок еще нет
- Проблема с подключением к БД

**Решение**:
1. Проверьте, что заявки есть в Supabase Table Editor
2. Нажмите "Обновить" в админ-панели
3. Проверьте браузер console (F12) на ошибки

### Форма не отправляет данные
**Причины**:
- Отсутствует обязательное поле
- Ошибка валидации
- Проблема с Supabase
- CORS ошибка

**Решение**:
1. Проверьте консоль браузера (F12)
2. Убедитесь, что все обязательные поля заполнены
3. Проверьте Supabase logs (Settings → Logs)

### Vercel показывает ошибку при деплое
**Причины**:
- Ошибка в коде
- Отсутствуют переменные окружения
- Проблема с зависимостями

**Решение**:
1. Проверьте Build logs в Vercel Dashboard
2. Убедитесь, что все переменные окружения добавлены
3. Попробуйте локально: `npm run build`

### Как посмотреть логи ошибок на продакшене?
1. **Vercel Logs**: Dashboard → Project → Deployments → Logs
2. **Supabase Logs**: Settings → Logs (в Supabase Dashboard)
3. **Browser Console**: F12 → Console
